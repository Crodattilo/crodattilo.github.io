Definizioni

Definizioni dei termini informatici.

HOME


🛈 Puoi cercare premendo Ctrl+F


Ads


ADS, è l'abbreviazione di "advertising" (annnci pubblicitari), molte applicazioni gratuite presenti su Play Store contengono annunci.

Apk


APK è l'estensione del file per un’applicazione Android. Tutte le applicazioni sono file APK. Questi sono files compressi che contengono le icone, i suoni e il codice sorgente dell'applicazione.

Backup


In informatica il backup (di riserva) è la copia di sicurezza dei dati, creata al fine di prevenirne la perdita.

BIOS


Il BIOS (Basic Input-Output System) è un insieme di software che fornisce una serie di funzioni di base per accedere ai componenti hardware del computer.

Bootloader


Bootloader (da "Boot" = Avvio e "Load" = Caricamento) è il processo che parte ancor prima dell'accensione del dispositivo e fornisce a quest'ultimo le istruzioni necessarie per avviare il sistema operativo.

Bootloop


Per bootloop (ciclo continuo di avvio) si intende il continuo blocco del dispositivo durante la procedura di avvio.

Bot


In informatica il Bot è un programma che svolge determinate operazioni automaticamente, comportandosi come un utente umano.

Browser


Il browser (navigatore) è un programma in grado di visualizzare risorse fornite da un server Web, ha spesso il mondo raffigurato nell'icona. Con il browser si può navigare in Internet.

Bug


Per bug (insetto) si intende un errore di funzionamento di un'app. Nella maggior parte dei casi un bug é dovuto a un errore nella scrittura del codice sorgente da parte del programmatore.

Cache


La cache è una sezione della memoria nella quale il computer immagazzina i dati usati più frequentemente, in modo da rileggerli più velocemente senza doverli ricaricare.

Client


Il client (cliente) è un programmma in grado di comunicare con un server, per un client il server è indispensabile per lo scambio di dati. Il browser è un client.

Codice sorgente


Il codice sorgente è un testo scritto dal programmatore con un linguaggio di programmazione, salvato come file sorgente.

CPU


La CPU (Central Processing Unit) chiamata anche processore è l'unità centrale di elaborazione, il suo compito è quello di gestire ed elaborare i dati.

Crash


In informatica per crash (schianto) si intende un blocco anomalo, improvviso, non richiesto di un processo in esecuzione o del sistema operativo.

Custom ROM


Una custom ROM è una versione modificata del sistema operativo.

Dialer


In informatica, il dialer (compositore) è un programma che, durante la navigazione in Internet, all'insaputa dell'utente attiva una connessione telefonica a pagamento.

Driver


In informatica, il driver (autista) è un programma che gestisce un tipo di periferica collegata al computer.

Firmware


Il firmware è l'insieme delle applicazioni e dei files di sistema presenti nella memoria.

Floppy disk


Il floppy disk (disco floscio) era un supporto di memorizzazione rimovibile, molto meno capiente delle periferiche odierne.

Font


In informatica, il font è un insieme di caratteri accomunati da uno stile grafico. Sono già presenti dei font nei vari sistemi operativi, tuttavia è possibile installarne altri. Un file font ha l'estensione .ttf (True Type Font).

Hard-brick


In informatica, il termine brick (mattone) viene usato per descrivere l'inutilità di un dispositivo che ha dei componenti hardware danneggiati e non può più funzionare

Hardware


In informatica, l'hardware (pezzo duro) è l'insieme di tutti i componenti fisici di un computer.

HTML


Il linguaggio HTML (Hyper Text Markup Language) viene usato per codificare le pagine web. Il browser è in grado di interpretare questo linguaggio.

Icona


In informatica l'icona è un'immagine a formato ridotto il cui scopo è rappresentare un programma, un'azione o un tipo di file.

Indirizzo IP


In informatica, l'indirizzo IP (Internet Protocol) è un numero che identifica un solo dispositivo (ognuno ne ha uno diverso) chiamato host.

Lag


Il lag (ritardo) é individuabile dal basso numero di FPS (frequenza dei fotogrammi) o dalla non risposta di un programma. Può verificarsi anche a causa di una connessione instabile.

Launcher


Il launcher é l'applicazione che lancia (apre) le altre app. Le schermate della home sono gestite dal launcher.

Malware


Con il termine malware (virus) si intende un programma o un file in grado di alterare il funzionamento o danneggiare un sistema operativo.

Patch


La patch (toppa) è una parte di software utile per migliorare un programma dal punto di vista della vulnerablità da parte dei malware.

Pdf


Il PDF (Portable Document Format), è un formato di file utile per convertire i documenti in modo che vengano visualizzati su tutti i dispositivi. I files PDF non possono essere modificati.

Periferica


Per periferica si intende un'apparecchio elettronico che può essere collegato esternamente ad un computer per ricevere e/o inviare dati.

Phishing


Il phishing è una truffa effettuata tramite un'e-mail, invitando il destinatario a fornire dati personali o denaro.

Plugin


In informatica, si chiama plugin (collegare) un programma non autosufficiente che collabora con un altro programma per aggiungerne ulteriori funzioni.

RAM


In informatica, la RAM (Random Access Memory) è la memoria di lavoro del computer dal contenuto modificabile che va perduto spegnendo il computer. È una memoria volatile.

Recovery


La recovery (recupero) è un software che permette di effettuare varie operazioni di sistema, come l’installazione di ROM o eseguire i backup delle impostazioni di Android. Si può accedere alla Recovery mode premendo anche il pulsante volume – insieme al pulsante power all'accensione del dispositivo (se presente usare anche il tasto home).

Risoluzione


In informatica, la risoluzione indica il grado di qualità di un'immagine o di un video.

ROM


In informatica, la ROM (Read Only Memory) è una memoria dal contenuto non modificabile che non va perduto spegnendo il computer. È una memoria permanente.

Screenshot


Lo screenshot (cattura) é una funzione del sistema operativo che permette di salvare una schermata come immagine.

SD


La SD (Secure Digital), è una scheda di memoria dalle dimensioni ridotte.

Server


In informatica, il server (fornitore) è un computer di elevate prestazioni che in rete fornisce un servizio agli altri dispositivi collegati chiamati client.

Sistema binario


Il sistema binario è il linguaggio dei computer digitali. Si tratta di un sistema numerico basato sui simboli 0 e 1, il sistema binario rappresenta lo stato di un circuito: 1→acceso; 0→spento.

Sistema operativo


Il sistema operativo, in informatica, è un insieme di programmi di sistema che gestiscono le risorse hardware e software del computer, fornendo servizi di base ai software applicativi (programmi installati dall'utente).

Soft-brick


In informatica, il termine brick (mattone) viene usato per descrivere l'inutilità di un dispositivo che ha il firmware corrotto e non può più funzionare.

Software


In informatica, il software (pezzo morbido) è l'insieme di tutti i programmi di un computer.

Spam


Spam è un messaggio pubblicitario non richiesto, inviato ripetutamente tramite posta elettronica.

Spyware


Uno spyware è un software scaricato inconsapevolmente durante la navigazione in Internet, programmato per inviare a terze parti i dati personali dell'utente.
